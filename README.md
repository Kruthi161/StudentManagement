SYSTEM REQUIREMENTS
------------------------------
1.SpringToolSuite
2.mysql Server
STEPS TO RUN THE PROJECT
----------------------------------
1.Right click on the project.
2.select Run As.
3.select SpringBootApp
4.Check the application in postman by using API end Points.
